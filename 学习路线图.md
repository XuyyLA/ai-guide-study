# 从零到 AI 大模型专家：完整学习路线

> 本路线融合 [AI-Guide-and-Demos](https://github.com/Hoper-J/AI-Guide-and-Demos-zh_CN) + [Hello-Agents](https://github.com/datawhalechina/hello-agents)，
> 以**理论→实操→项目**三层递进，每步有明确产出和检验标准。
>
> 设计原则：
> - **快速出活**：第 1 周就能调 API 做应用，建立信心
> - **理论后置**：先会用，再理解为什么，避免一上来就啃数学
> - **交叉喂养**：Guide（模型能力）和 Agents（智能体架构）交替推进
> - **项目驱动**：每阶段末尾有完整项目，把零散知识串起来

---

## 全局路线图

```
Week 1-2  ┃ Phase 0: 破冰 ┃──── API 调用 + Gradio 应用（立刻能用）
Week 3-4  ┃ Phase 1: 入理 ┃──── Prompt 工程 + 生成策略（理解模型行为）
Week 5-6  ┃ Phase 2: 识模 ┃──── AutoModel + 部署 + DL 基础补课
Week 7-8  ┃ Phase 3: 调模 ┃──── 微调(LoRA) + 量化 + Agent 范式
Week 9-10 ┃ Phase 4: 建体 ┃──── RAG + Agent 框架 + 上下文工程
Week 11-12┃ Phase 5: 成器 ┃──── 多智能体 + 评估 + 毕业项目
```

---
