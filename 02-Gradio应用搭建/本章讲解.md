## 本章知识

### 1. Gradio 框架

Gradio 是快速构建 AI 演示界面的 Python 库，几行代码即可创建 Web UI：

```python
import gradio as gr
gr.Interface(fn=处理函数, inputs=输入组件, outputs=输出组件).launch()
```

### 2. 常用组件

| 组件 | 用途 |
|------|------|
| `gr.Textbox()` | 文本输入/输出 |
| `gr.Chatbot()` | 聊天界面（多轮对话） |
| `gr.Dropdown()` | 下拉选择 |
| `gr.Slider()` | 数值滑块 |
| `gr.Markdown()` | Markdown 渲染 |

### 3. Gradio 6.x 注意事项

- Chatbot 回调传入 `dict` 格式 `{"role": "user", "content": "..."}`，不再用 `tuples`
- 事件绑定用 `.click()` / `.change()`，`.input()` 不稳定
- 回调返回组件实例可动态更新 UI（如 Slider 范围）

## 代码讲解

| 文件 | 说明 |
|------|------|
| `config.py` | 同 01 的 Config 类，智谱 API 配置 |
| `01文章摘要.py` | 单轮：用户输入文章 → LLM 生成 ~100 字摘要 → Textbox 显示 |
| `02角色扮演.py` | 单轮：system prompt 设定为"林黛玉"角色 → 按角色风格回复 |
| `03定制化任务.py` | 单轮：用户描述任务 → LLM 分析并给出 2-3 个具体执行策略 |

三个应用都是**单轮对话**模式：每次请求独立，不保留上下文。

## 运行方式

python 01文章摘要.py    # 浏览器自动打开 http://localhost:7860
