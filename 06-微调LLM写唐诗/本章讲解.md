## 本章知识

### 1. 微调流程

```
加载预训练模型 → 注入 LoRA 适配器 → 准备训练数据 → SFTTrainer 训练 → 保存 checkpoint
→ 加载 checkpoint → 推理对比
```

### 2. BitsAndBytes 4-bit 量化

将模型权重从 FP32 压缩到 4-bit，大幅减少显存占用（7B 模型从 ~28GB 降到 ~5GB）：

```python
BitsAndBytesConfig(
    load_in_4bit=True,
    bnb_4bit_quant_type="nf4",          # NormalFloat4 量化
    bnb_4bit_use_double_quant=True,      # 双重量化进一步压缩
    bnb_4bit_compute_dtype=torch.bfloat16  # 计算时用 bfloat16
)
```

> ⚠️ 需要 NVIDIA GPU + CUDA 版 PyTorch

### 3. LoRA 配置

```python
LoraConfig(
    r=8,              # 低秩矩阵维度
    lora_alpha=16,    # 缩放因子
    target_modules=[...],  # 要微调的层
    lora_dropout=0.05,
    task_type="CAUSAL_LM",
)
```

LoRA 只训练 `r × (输入维度 + 输出维度)` 个参数，而非全部权重。

### 4. completion_mask 机制

训练时只在**模型输出部分**计算 loss，prompt 部分不参与：

```
[prompt tokens: 0, 0, 0, ...] [output tokens: 1, 1, 1, ...]
```

`DataCollatorForLanguageModeling(completion_only_loss=True)` 根据 mask 将 prompt 位置的 labels 设为 -100（忽略）。

### 5. ChatML 格式（Qwen）

```python
messages = [
    {"role": "system", "content": "你是一个乐于助人的助手且擅长写唐诗。"},
    {"role": "user", "content": "以下是一首唐诗的第一句话..."},
]
prompt = tokenizer.apply_chat_template(messages, tokenize=False, add_generation_prompt=True)
```
