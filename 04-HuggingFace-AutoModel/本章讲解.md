## 本章知识

### 1. Transformers pipeline API

一行代码完成 NLP 任务：

```python
from transformers import pipeline
result = pipeline("任务类型", model="模型名")(输入文本)
```

### 3. AutoModel vs pipeline

- **pipeline**：高层 API，自动组装 tokenizer + model + 后处理，适合快速验证
- **AutoModel**：底层 API，手动加载 tokenizer 和 model，灵活控制推理参数

### 4. 中文模型 vs 英文模型

中文任务需选择中文预训练模型（如 `bert-base-chinese`），英文模型对中文效果差。

## 代码讲解

| 文件 | 核心逻辑 |
|------|---------|
| `01文本生成.py` | `pipeline("text-generation")` + 中文 GPT-2，生成续写文本 |
| `02填空任务.py` | `pipeline("fill-mask")` + BERT，预测 `[MASK]` 位置的词（top-5） |
| `03机器翻译.py` | `pipeline("translation")` + OPUS-MT，中→英→中往返翻译 |
| `04问答系统.py` | `pipeline("question-answering")` + RoBERTa，从上下文抽取答案 |
| `05命名实体识别.py` | `pipeline("ner")` + BERT-NER，识别人名/地名/组织名 |
| `06文本分类.py` | `pipeline("text-classification")` + RoBERTa，新闻情感分类 |
| `07特征提取.py` | `pipeline("feature-extraction")` + BERT，输出隐藏层向量 |
| `08查看源码.py` | `inspect.getsource(AutoModel)`，查看类源码理解实现 |
