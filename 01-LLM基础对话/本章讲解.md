## 本章知识

### 1. OpenAI 兼容 API 调用

智谱 GLM 系列提供 OpenAI 兼容接口，使用 `openai` 库直接调用：

```python
from openai import OpenAI
client = OpenAI(api_key="...", base_url="https://open.bigmodel.cn/api/paas/v4")
```

核心方法：`client.chat.completions.create(model=..., messages=...)`

### 2. 消息格式

`messages` 列表中每条消息包含 `role` 和 `content`：
- `system`：系统提示，定义模型角色和行为
- `user`：用户输入
- `assistant`：模型回复

### 3. 多轮对话

关键在于**累积 messages 列表**，每次请求都带上完整历史，模型才能理解上下文。

### 4. 流式输出

设置 `stream=True`，返回生成器而非一次性结果。逐个 chunk 读取并打印（`flush=True` 实现实时显示）。

## 代码讲解

| 文件 | 说明 |
|------|------|
| `config.py` | `Config` 类集中管理 API_KEY、BASE_URL、MODEL，`get_client()` 创建客户端 |
| `01初次对话.py` | 单轮对话：发送一条 user 消息，打印 assistant 回复 |
| `02多轮对话.py` | `ChatBot` 类：`chat()` 累积 messages，`forever_chat()` 循环输入直到"退出" |
| `03流式输出.py` | `stream=True` + 遍历 chunks + `flush=True` 逐字打印 |

## 关键 API 参数

```python
client.chat.completions.create(
    model="glm-4-flash",       # 模型名称
    messages=[...],            # 消息列表
    stream=True,               # 流式输出
    temperature=0.7,           # 随机度（0-1）
    max_tokens=1024,           # 最大生成 token 数
)
```
